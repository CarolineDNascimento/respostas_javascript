Respostas sobre JavaScript

1. JavaScript e Java são a mesma coisa?

Não, JavaScript e Java são duas linguagens de programação diferentes. Java é uma linguagem de programação orientada a objetos, enquanto JavaScript é uma linguagem de script usada para desenvolvimento web, principalmente no lado do cliente (frontend).

2. Quando surgiu o JavaScript?

O JavaScript foi criado em 1995 por Brendan Eich, enquanto ele trabalhava na Netscape Communications Corporation. Inicialmente, a linguagem foi chamada de "LiveScript".

3. Quem foi o responsável pela criação do JavaScript?

Brendan Eich, um programador da Netscape, é o responsável pela criação do JavaScript.

4. Qual o caminho trilhado pelo responsável pela criação do JavaScript, trata-se de alguma empresa ainda no mercado?

Brendan Eich trabalhou na Netscape, empresa que desempenhou um papel importante na popularização da web. Após sua passagem pela Netscape, Eich cofundou a Mozilla, a fundação responsável pelo desenvolvimento do navegador Firefox. A Mozilla ainda está ativa e continua no mercado com produtos como o Firefox e outras iniciativas de código aberto.
